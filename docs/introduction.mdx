##### What is LowCSS?

**LowCSS** is a utility-first framework that provides a collection of small functional CSS blocks that you can use for building complex and responsive components without writting a single line of CSS code.

Each utility block can be configured for being applied on different conditions, for example to respond to different screen sizes or to respond to some user actions like hover or focus.

##### Fundamentals

To undestand how **LowCSS** works and it's benefits, let's have a look to the following example.

Traditionally, to style an HTML component you might use custom CSS written in a separate `.css` file or directly in your HTML in a `<style>` tag.

<ExampleCode className="d:flex items:center justify:center">
    <div className="d:flex gap:4 bg:white r:lg shadow:md p:6 w:full maxw:sm">
        <img className="r:full w:16 h:16" src="https://i.pravatar.cc/120?u=123" />
        <div className="w:full">
            <div className="font:bold">Robin Doe</div>
            <div className="text:sm text:gray-600">Product designer at <strong>@company</strong>.</div>
            <div className="text:xs text:gray-400 mt:1">robin.doe@company.com</div>
        </div>
    </div>
</ExampleCode>

```html
<div class="card">
    <img class="card-image" src="https://i.pravatar.cc/120?u=123">
    <div class="card-content">
        <div class="card-content-title">Robin Doe</div>
        <div class="card-content-subtitle">
            Product designer at <strong>@company</strong>.
        </div>
        <div class="card-content-mail">robin.doe@company.com</div>
    </span>
  </div>
</div>

<style>
    .card {
        background-color: #ffffff;
        border-radius: 0.75rem;
        box-shadow: 0 0.5rem 1rem -0.25rem rgba(0,0,0,.2), 0 0 0 1px rgba(0,0,0,.02);
        display: flex;
        gap: 1rem;
        max-width: 24rem;
        padding: 1.5rem;
        width: 100%;
    }
    .card-image {
        border-radius: 100%;
        height: 4rem;
        width: 4rem;
    }
    .card-content {
        width: 100%;
    }
    .card-content-title {
        font-weight: 700;
    }
    .card-content-subtitle {
        color: #4b5563;
        font-size: 0.875rem;
    }
    .card-content-mail {
        color: #a4abb6;
        font-size: 0.75rem;
        margin-top: 0.25rem;
    }
</style>
```

But when styling the elements with **LowCSS**, you can use the pre-existing utility classes directly in the HTML instead:

```html
<div class="d:flex gap:4 bg:white r:lg shadow:md p:6 w:full maxw:sm">
    <img class="r:full w:16 h:16" src="https://i.pravatar.cc/120?u=123" />
    <div class="w:full">
        <div class="font:bold">Robin Doe</div>
        <div class="text:sm text:gray-600">
            Product designer at <strong>@company</strong>.
        </div>
        <div class="text:xs text:gray-400 mt:1">robin.doe@company.com</div>
    </div>
</div>
```

With **LowCSS** you can style your entire HTML without writting a single line of CSS code.
