Perfect for creating navigation bars, button groups, or any horizontal layout. {{/prose-paragraph.html}} {{>>prose-code.html language="css" label="index.css" icon="file-code"}} .hstack { align-items: center; align-self: stretch; display: flex; flex-direction: row; } {{/prose-code.html}} {{>>preview.html}}
