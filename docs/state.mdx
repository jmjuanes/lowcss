You can conditionally apply utilities adding a modifier to the utility class with the target condition that you want to apply.

###### Hover, focus and active

Apply utilities on hover, focus and active using the `:hover`, `:focus` and `:active` modifiers. For example, to change the background color only when the user hovers the element, just append the `:hover` modifier to the utility class.

<ExampleCode className="d:flex justify:center">
    <div className="bg:blue-500 bg:blue-800:hover text:white text:sm px:4 py:2 r:md cursor:pointer">
        <strong>Hover me</strong>
    </div>
</ExampleCode>

```html
<div class="bg:blue-500 bg:blue-800:hover [...]">
    Hover me
</div>
```

###### Group modifiers

You can also apply utilities based on the state of the parent element. Just do the following:
- Add the `group` class to the parent element.
- Apply the utilities to the child elements when the user hover or focus the parent element using the `:group-hover` or `:group-focus` modifiers.

<ExampleCode className="d:flex justify:center">
    <div className="group d:flex items:center justify:center bg:white position:relative r:md py:2 px:4 b:1 b:solid b:gray-200">
        <span className="text:lg text:gray-600">
            <Icon icon="bars" />
        </span>
        <span className="text:gray-600 ml:2">Menu</span>
        <div className="d:none d:block:group-hover position:absolute w:48 left:0 top:full pt:1">
            <div className="bg:white b:1 b:gray-300 b:solid r:md d:flex flex:col gap:1 py:1">
                <div className="px:4 py:2 bg:gray-200:hover cursor:pointer">Dashboard</div>
                <div className="px:4 py:2 bg:gray-200:hover cursor:pointer">Profile</div>
                <div className="px:4 py:2 bg:gray-200:hover cursor:pointer">Settings</div>
                <div className="h:0 w:full b:1 b:gray-200 b:solid" />
                <div className="px:4 py:2 bg:gray-200:hover cursor:pointer">Sign out</div>
            </div>
        </div>
    </div>
</ExampleCode>

```html
<div class="group [...]">
    <!-- button content -->
    <div class="d:none d:block:group-hover [...]">
        <!-- dropdown items -->
    </div>
</div>
```
