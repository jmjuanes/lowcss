**LowCSS** includes a default color palette that can be used with utilities such as `background-color` and `border`. 

<table className="w:full b:1 b:solid b:gray-200">
{Object.keys(props.data.colors).map(key => (
    <tr key={key} className="b:1 b:solid b:gray-200">
        <td className={`bg:${key} p:4`} />
        <td className="p:4 bl:1 bl:solid bl:gray-200">
            <span className="font:mono text:indigo-700">{key}</span>
        </td>
        <td className="p:4 bl:1 bl:solid bl:gray-200">
            <span className="font:mono text:indigo-700">{props.data.colors[key]}</span>
        </td>
    </tr>
))}
</table>
